# cursos
repositórios de cursos online

## comandos do git

copiar o repositorio para o ambiente local
> git clone https://github.com/Thyoliver32/cursos

definir email do git 
> git config --global user.email "you@example.com"

definir usuario do git
> git config --global user.name "Your Name"

criar ultra branch
> git checkout -b develop

verificar alterações 
> git status

preparar o comit
>git add .

comitar o codigo
>git commit -m "meu primeiro commit"

enviar o commit para o repositorio 
>git push origin main
